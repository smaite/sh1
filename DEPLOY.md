# Quick Start Guide

## Running Locally

1. Open terminal in this folder (D:\sh1)
2. Run: `npm install` (first time only)
3. Run: `npm start`
4. Open browser: http://localhost:3000

## Deploying to Netlify

### Option 1: Drag & Drop (Easiest)
1. Open the `public` folder
2. Drag all files to Netlify deploy page
3. Done!

### Option 2: Git + Netlify (Recommended)
1. Create GitHub repo
2. Upload the `public` folder contents
3. Connect repo to Netlify
4. Deploy automatically!
